# EEG Workshop Haifa

This project is for the EEG workshop in Haifa.

## Project Standards
- Language: Python (prefer marimo notebooks for analysis)
- Libraries: MNE-Python, marimo, dash, eegdash, NumPy, Matplotlib, Scikit-learn
- Documentation: Use Docstrings for functions and Markdown cells in notebooks.

## Environment & Dependencies
This project uses `uv` for dependency management.
- Python: `3.9` to `3.11`
- Core: `numpy`, `scipy`, `pandas`, `matplotlib`
- EEG: `mne`, `mne-bids`, `autoreject`, `pyvista`
- UI: `dash`, `flask`, `plotly`, `eegdash`
- Notebooks: `marimo`, `jupyter`, `jupyterlab`

## Context
- Focus on EEG signal processing, artifact removal, and feature extraction.
- Data is likely in `.edf`, `.bdf`, or `.fif` formats.
